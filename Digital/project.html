1. SMART TRASH BIN :

Domain : Internet of Things
Status : Finished
Start : February 2019
End : May 2019

Hardwares Used :

  • Ultrasonic level sensor : HC-SR04
  • Load cell weight sensor : 10kg
  • Smell sensor : MQ135
  • Raspberry Pi3
  • Microcontroller
  • OS : Raspbian
  • Language : Python3

Description :

Smart Trash Bin is a garbage monitoring system, monitors the weight , level and status of the garbage. Ultrasonic sensor detects the level of the garbage , load cell monitors the weight and smell sensor detects the unpleasant smells. We fix the threshold value for identifying whether the garbage is full or not. If it reaches that value it sends message to the authorized person so that he can clean the garbage before it overflows. This helps in preventing the pollution and also the time and workload of the workers.

2. DIGITAL MENU FOR RESTAURANTS :

Domain : Web Technology
Status : Gathering requirements
Start : September 2019
End : -

Technologies Used :

  • HTML 5
  • CSS 3
  • Javascript
  • QR Code
  • QR Scanner (in mobile phones)

Description :

Digital Menu for Restaurants is a food ordering system. This contains the menu items of the restaurant. Sometimes waiter forget to note our food items that time we have to wait. We eliminated this problem by ordering food items of our own with the help of our project. This project help the customer to order their food items by scanning the QR code of the table with specified table number. Once they order their items it asks for recheck of our items. When we ordered the items after checking it bill will also be generate. The bill will be display in the Restaurant Admin page by that they can deliver the items to the corresponding table number and it also gives the solution for the another problem that is the waiter might deliver it to the wrong tables also this is also prevented. The status of the bill will also be displayed in the admin page. This will make customers happy.
